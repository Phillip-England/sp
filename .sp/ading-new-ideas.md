To add a new idea you can either type the idea directly or import it from the clipboard

typing an idea directly will be done using 'sp "some title" "some text for title and idea"' which will add their idea manually

you can also do sp "another title" --cb and thatll take anything on the clipboard and use it as a new idea

Keep in mind that "some title" will become "some-title" in the markdown files name
